В скрипте используются REST API запросы:
- /configure/groups  — список групп МС
- /configure/channels/  —  список камер
- /configure/groups/guid_группы_пользователей  — доступные камеры в guid группы пользователей
 
Работает с net6.0 — т.е. для использования не требуется дополнительно устанавливать компоненты, если установлен МС.
*Работает только на ENT/ULTRA
